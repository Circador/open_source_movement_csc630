Resources Assignments git cheatsheet
  • git init
  • git status
  • git add <filename>
  • git rm <filename>
  • git reset <filename>
  • git commit
  • git diff
  • git log
  • git checkout: Change the workspace (i.e. the current files in the directory) to those located at the specified commit. In essence, moves "you" (a.k.a. HEAD) to the commit. If you add the -b flag, it will create a new branch with the specified name: git checkout -b <new_branch_name>.
  • git branch <branchname>
  • git remote manages your set of tracked repositories. remote has many different ways to be called. The most commonly used are:
    • git remote add <name> <url> adds a remote named <name> for the repository at <url>.
    • git remote rename <old> <new> renames the remote named <old> to <new>
    • git remote show <name>
      • gives information about the remote <name>.
    • git remote remove <name> removes the remote named <name>.
    These are fairly simplified commands. For further information on their usage, visit remote's documentation.

  • git merge <branchname>
  • git fetch
  • git pull <remote_repo> <remote_branch>
